# vvtab
vvtab是一款开源的组件式标签页浏览器扩展，我希望抛砖引玉让更多有兴趣的伙伴一起来完善它。(vvtab is an open source component-based browser extension of the tabbed page. I hope that it could set an example and inspire more interested partners to improve it together.)
